# 🚀 Bootstrap Demo Project

A modern and responsive **startup landing page** built using **Bootstrap 5**. This project demonstrates how to create a clean UI with sections like hero, features, pricing, testimonials, and contact form.

---

## 📌 Features

* ✅ Responsive Navbar with smooth navigation
* ✅ Hero section with call-to-action buttons
* ✅ Feature cards with icons
* ✅ Pricing section with multiple plans
* ✅ Testimonials section
* ✅ Contact form UI
* ✅ Footer with social media links
* ✅ Fully responsive design using Bootstrap Grid
Live demo- https://glistening-unicorn-e5a521.netlify.app/
---

## 🛠️ Tech Stack

* HTML5
* CSS3
* Bootstrap 5
* Bootstrap Icons
